MULTIMC(1)
==========
:doctype: manpage


NAME
----
ultimmc - a launcher and instance manager for Minecraft.


SYNOPSIS
--------
*ultimmc* ['OPTIONS']


DESCRIPTION
-----------
UltimMC is a custom launcher for Minecraft that allows you to easily manage
multiple installations of Minecraft at once. It also allows you to easily
install and remove mods by simply dragging and dropping.
It is a light fork of MultiMC with ElyBy and local (cracked) account support.
Here are the current features of UltimMC.

OPTIONS
-------
*-d, --dir*='DIRECTORY'::
    Use 'DIRECTORY' as the UltimMC root.

*-l, --launch*='INSTANCE_ID'::
    Launch the instance specified by 'INSTANCE_ID'.

*--alive*::
    Write a small 'live.check' file after UltimMC starts.

*-h, --help*::
    Display help text and exit.

*-v, --version*::
    Display program version and exit.

EXIT STATUS
-----------
*0*::
    Success

*1*::
    Failure (syntax or usage error; configuration error; unexpected error).
